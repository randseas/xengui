Cross-platform clipboard library in rust.

## Supported platforms

- Windows
- Wayland
- X11
- macOS (planned)
- Android (planned)
- iOS (planned)
- WASM

## Example

```rust
let a = "b";
```

## Installation

`Cargo.toml`

```toml
[dependencies]
xen-clipboard = "0.1.0"
```
